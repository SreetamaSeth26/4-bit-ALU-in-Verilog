# 🔢 4-bit ALU in Verilog

A fully functional **4-bit Arithmetic Logic Unit (ALU)** implemented in Verilog, supporting 16 operations including arithmetic, logic, shift, rotate, and comparison — with a complete testbench.

---

## 📐 Module Overview

| Parameter | Details |
|-----------|---------|
| **Inputs** | `A [3:0]`, `B [3:0]`, `sel [3:0]` |
| **Output** | `result [4:0]` |
| **Operations** | 16 (selected via 4-bit `sel`) |

The 5-bit output accommodates carry/overflow for arithmetic results (e.g., `ADD`, `INCREMENT`).

---

## ⚙️ Supported Operations

| `sel` | Operation | Description |
|-------|-----------|-------------|
| `4'b0000` | ADD | `A + B` |
| `4'b0001` | SUB | `A - B` |
| `4'b0010` | AND | `A & B` |
| `4'b0011` | OR | `A \| B` |
| `4'b0100` | XOR | `A ^ B` |
| `4'b0101` | NOT A | `~A` |
| `4'b0110` | NAND | `~(A & B)` |
| `4'b0111` | NOR | `~(A \| B)` |
| `4'b1000` | XNOR | `~(A ^ B)` |
| `4'b1001` | Logical Left Shift | `A << 1` |
| `4'b1010` | Logical Right Shift | `A >> 1` |
| `4'b1011` | Rotate Left | Rotate `A` left by 1 bit |
| `4'b1100` | Rotate Right | Rotate `A` right by 1 bit |
| `4'b1101` | Increment | `A + 1` |
| `4'b1110` | Decrement | `A - 1` |
| `4'b1111` | Compare | `1` if `A > B`, else `0` |

## 🚀 Getting Started

### Simulation — EDA Playground

1. Go to [EDA Playground](https://www.edaplayground.com/)
2. Paste `alu.v` and `tb.v` into the editor panes
3. Select **Icarus Verilog** as the simulator
4. Check **Open EPWave after run** to view the waveform from `dump.vcd`
5. Hit **Run**

### Netlist — Yosys (Terminal)

Generate a synthesized netlist using the open-source [Yosys](https://yosyshq.net/yosys/) synthesis suite:

```bash
# Launch Yosys
yosys

# Inside the Yosys shell:
read_verilog alu.v          # Load the design
synth -top alu              # Synthesize with 'alu' as top module
show                        # View schematic (requires xdot)

# Export netlist
write_verilog alu_netlist.v          # Verilog netlist
write_json alu_netlist.json          # JSON netlist (for netlistsvg, etc.)
```

Or as a one-liner script:

```bash
yosys -p "read_verilog alu.v; synth -top alu; write_verilog alu_netlist.v"
```

## 🧪 Testbench

The testbench (`tb.v`) exercises all 16 operations sequentially, each separated by a 10-unit time delay. Results are printed via `$monitor` in binary format:

```
A=0101 B=0011 sel=0000 || Result=01000   ← ADD: 5 + 3 = 8
A=1001 B=0011 sel=0001 || Result=00110   ← SUB: 9 - 3 = 6
A=1100 B=1010 sel=0010 || Result=01000   ← AND
...
```

A VCD dump file (`dump.vcd`) is generated for waveform analysis.

---

## 💡 Design Notes

- The ALU uses a **combinational `always @(*)` block** with a `case` statement for clean, readable operation selection.
- The **5-bit output** prevents truncation on operations that can overflow a 4-bit result (addition, increment).
- **Rotate operations** are implemented using bit-slicing and concatenation rather than shift operators, correctly wrapping the shifted-out bit back in.
- The **compare operation** (`sel = 4'b1111`) returns `5'b00001` if `A > B`, and `5'b00000` otherwise.
